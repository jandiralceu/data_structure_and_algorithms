# Data Structure and Algorithms

This repository contains many data structure implementations using Typescript.
Navigate over the `src` folder to see all data structure implementations.

## Why this repository?

This repository is a guide of how to implement some data structure in typescript, that is a knowledge required in interviews even our your daily work.

## How to build?

Ensure that you have the `node v16.13.2` or above, `npm` or `yarn` .

```
yarn build
```

or

```
npm run build
```

## How to run each example?

First, build the project running the command above. Once builded, open one folder inside the build generated file(dist).

Example:

```
cd dist/01_linked_list/01_singly
```

Each data structure has his own implementation file, that file, has a suffix `impl.js`. In the folder above is `linked_list.impl.js`, for instance.

Use terminal to run this folder

```
node linked_list.impl.js
```
